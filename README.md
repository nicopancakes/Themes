# OWO++
A Modificiation Manager for SDR++

-----

# How to Install Themes:

Find your OWO++ Folder,

Go to root\res\themes

Download the Theme <img width="1122" height="725" alt="image" src="https://github.com/user-attachments/assets/25742cd8-cf81-447c-8fdc-4bb7af4ad7b4" />


Drag it into \themes Folder.

Close and Reopen OWO++ and Check Themes.
